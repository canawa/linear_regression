О линейной регрессии

Линейная регрессия — это статистический метод, используемый для моделирования линейной зависимости между зависимой переменной (целевая переменная) и одной или несколькими независимыми переменными (признаки).

Принцип работы:

  • Алгоритм находит наилучшую прямую линию, которая минимизирует сумму квадратов ошибок между предсказанными и реальными значениями
  • Используется метод наименьших квадратов для нахождения оптимальных коэффициентов
  • Качество модели оценивается с помощью среднеквадратичной ошибки (MSE)
Важно: Ваш CSV файл должен содержать колонку с именем target, которая будет использоваться как целевая переменная. Все остальные колонки будут рассматриваться как признаки для модели.

Требования к данным:

  • Файл в формате CSV
  • Наличие колонки target с числовыми значениями
  • Остальные колонки должны содержать числовые признаки
  • Отсутствие пропущенных значений (NaN)
